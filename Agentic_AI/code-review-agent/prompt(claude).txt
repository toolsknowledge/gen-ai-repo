I am a software engineering student learning Agentic AI development.

Build me a complete, production-ready Code Review Agent application from scratch with the following requirements:

WHAT TO BUILD:
A full-stack AI-powered code review tool where a user pastes or uploads code, and an AI agent automatically reviews it for bugs, security issues, performance problems, and code quality.

TECH STACK:
- Backend: Python + FastAPI + Anthropic Claude API (claude-opus-4-5)
- Frontend: React + Vite (JavaScript, no TypeScript)
- AI: Anthropic Claude with tool calling (ReAct agent pattern)

ARCHITECTURE:
- Backend runs on port 8000 (FastAPI + uvicorn)
- Frontend runs on port 5173 (Vite dev server)
- React calls FastAPI via HTTP POST
- FastAPI runs the agent loop and calls Anthropic API

BACKEND REQUIREMENTS (server.py):
1. FastAPI app with CORS enabled for localhost:5173
2. Anthropic client initialized from ANTHROPIC_API_KEY environment variable
3. Two tools Claude can call:
   - analyze_complexity: counts lines, functions, max nesting depth, cyclomatic complexity score
   - check_security_patterns: scans for hardcoded passwords, SQL injection, eval(), pickle, MD5, shell=True, HTTP links, DEBUG=True using regex
4. A ReAct agent loop (max 8 iterations) that:
   - Sends code to Claude with the tools
   - Executes any tools Claude requests
   - Passes tool results back to Claude
   - Repeats until Claude returns a final review
5. Three endpoints:
   - GET /health → returns api_key_set status
   - POST /review → runs the full agent, returns review + metrics + security
   - POST /scan → runs local tools only, no Claude call, instant response
6. System prompt that instructs Claude to always use both tools first, then write a structured markdown review covering: Overview, Bugs, Security, Performance, Readability, What's Done Well, Priority Fixes, and a Score table
