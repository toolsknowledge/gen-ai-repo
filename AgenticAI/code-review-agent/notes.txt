
Step 1.
    clone master url in desktop

    > git clone https://github.com/toolsknowledge/gen-ai-repo.git

    Note : "gen-ai-repo" directory will create automatically in "Desktop"

Step 2.
    open with vscode (gen-ai-repo)

Step 3.
     checkout branch
     > terminal
     > git checkout batch-20

Step 4.
        navigate to coding-review-agent-backend

        > cd Agentic_AI
        > cd code-review-agent
        > cd code-review-agent-backend

Step 5.
        python3 -m venv venv
             (or)
        python -m venv venv

Step 6.
        source venv/bin/activate (mac)
        venv\Scripts\activate (windows)

Step 7.
        pip install -r requirements.txt

Step 8.
        set API key

     
Step 9.
        start uvicorn server
        > uvicorn server:app --reload

Step 10.
    navigate to code-review-frontend
        > cd Agentic_AI
        > cd code-review-agent
        > cd code-review-frontend

Step 11.
        npm install

Step 12.
        npm run dev
