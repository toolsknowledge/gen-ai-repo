Variable
********
    "Variables" are used to "store the data"
    Ex.
            int
            string
            boolean
            ---
            ---
            ---
    datatype representing "type of data"
    python supports below datatypes
    1) int
    2) str
    3) bool
    4) list
    5) tuple
    6) dict
    7) set
    8) None

print() is the "predefined function", used to display "data on console"

"rich" library, used to customize the console outputs

"pip" is the tool, used to download the libraries

> pip install rich
